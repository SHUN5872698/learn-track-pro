## docs
```bash
.gemini/docs/
├── apis/                  # API仕様、エンドポイント、リクエスト/レスポンス例
├── architectures/         # システム設計、コンポーネント図
├── cookbooks/             # プロジェクト固有の開発手順やレシピ
├── databases/             # データベーススキーマ、ER図、マイグレーション戦略
├── decisions/             # 設計上の意思決定の記録 (ADR)
├── designs/               # UI/UXデザイン、ワイヤーフレーム、モックアップ、デザインシステム
├── development-processes/ # 開発ワークフロー、コーディング規約、一般的な開発プロセスチェックリスト
├── infrastructures/       # デプロイ、サーバー設定、Docker、CI/CD、監視
├── project-infos/         # プロジェクトロードマップ、一般的なプロジェクト情報、貢献ガイド
├── securities/            # 認証・認可、CSRF/XSS対策、セキュリティベストプラクティス
├── tasks/                 # プロジェクト固有の具体的なタスクリスト、機能実装チェックリスト
└── tests/                 # テスト戦略、単体/結合/E2Eテストガイドライン、テストレポート
```
