## Q1. 学習コストの評価

### Page Visibility API

**学習コスト: ★☆☆☆☆（非常に低）**

**理由:**

- APIは非常にシンプル（document.visibilityStateの監視のみ）
- 実装コード量: 10-20行程度
- 学習時間: 1-2時間で十分
- ブラウザ互換性: モダンブラウザは95%以上対応
- **実装時期**: ストップウォッチ機能と同時（Week 2-3日目の進捗70%以上で判断）

```jsx
// 実装例（シンプル）
document.addEventListener('visibilitychange', () => {
  if (document.hidden) {
    // タブ非表示時の処理
    pauseTimer();
  } else {
    // タブ表示時の処理
    resumeTimer();
  }
});

```

### Chart.js

**学習コスト: ★★☆☆☆（低）**

**理由:**

- Vue.js用ラッパー（vue-chartjs）が充実
- 公式ドキュメントが詳細
- 基本的な棒グラフなら数時間で実装可能
- 学習時間: 4-8時間程度
- **実装時期**: Week 3バックエンド実装時に並行学習

- **学習時期**: 事前準備期間（8月19日〜21日のAPI仕様設計時）

**学習すべき範囲:**

**必須機能:**

**応用機能（時間があれば）:**
