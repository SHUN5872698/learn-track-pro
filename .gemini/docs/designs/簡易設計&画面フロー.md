```

│ ✏️ 編集      │
│ ℹ️ 詳細      │

└──────────────┘

┌──────────────┐
│ ✅ 完了にする │ <- 追加
```

### 画面遷移フロー

```mermaid
flowchart TD
    A[学習内容一覧] --> B[カードクリック]
    B --> C[学習記録作成]

    A --> D[三点メニュー > 詳細]
    D --> E[学習内容詳細]

    E --> F[セクションクリック]
    F --> G[セクション別記録一覧]

    G --> H[編集ボタン]
    H --> I[学習記録編集]

    G --> J[削除ボタン]
    J --> K[削除確認モーダル]

    G --> L[記録を追加]
    L --> C

    E --> M[学習を記録する]
    M --> C

    I --> |保存| G
    K --> |削除実行| G

```
