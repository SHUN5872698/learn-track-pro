### 画面遷移フロー

```mermaid
flowchart TD
    A[学習内容一覧] --> B[カードクリック]
    B --> C[学習記録作成]

    A --> D[三点メニュー > 詳細]
    D --> E[学習内容詳細]

    E --> F[セクションクリック]
    F --> G[セクション別記録一覧]

    G --> H[編集ボタン]
    H --> I[学習記録編集]

    G --> J[削除ボタン]
    J --> K[削除確認モーダル]

    G --> L[記録を追加]
    L --> C

    E --> M[学習を記録する]
    M --> C

    I --> |保存| G
    K --> |削除実行| G

```

## 画面フロー

```mermaid
flowchart TD
    A[ログイン] --> B[学習内容一覧<br/>ダッシュボード]

    %% 学習内容管理
    B --> C[カードクリック]
    C --> REC[学習記録作成]

    B --> T[三点メニュー]
    T --> ED[編集選択]
    T --> DE[詳細選択]
    T --> DL[削除選択]
    T --> CO[完了/再開選択]

    ED --> G[学習内容編集画面]
    DE --> D[学習内容詳細画面]
    DL --> DC[削除確認モーダル]
    CO --> CS[完了/再開処理]

    B --> O[新規作成ボタン]
    O --> F[学習内容登録画面]

    %% 詳細画面からの遷移
    D --> SEC[セクションクリック]
    SEC --> RLIST[セクション別記録一覧]

    RLIST --> REDIT[編集ボタン]
    REDIT --> REDITED[学習記録編集]

    RLIST --> RDEL[削除ボタン]
    RDEL --> RDELCONF[削除確認モーダル]

    RLIST --> RADD[記録を追加]
    RADD --> REC

    D --> DREC[学習を記録する]
    DREC --> REC

    %% レポート関連
    B --> NAV_REPORT[サイドメニュー<br/>レポート]
    NAV_REPORT --> I[レポート画面]
    I --> J[学習推移詳細]

    %% ユーザー管理
    B --> USER[ヘッダー<br/>ユーザー名]
    USER --> M[プロフィール]
    M --> N[ユーザー編集]

    %% ログアウト
    B --> LOGOUT[サイドメニュー<br/>ログアウト]
    LOGOUT --> A

```
