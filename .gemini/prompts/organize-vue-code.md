@.gemini/docs/development-processes/vue-coding-standards.md の「必須の順序ルール」セクションに従って、コードを整理してください。

### 要件
1. 「順序チェックリスト」の順番でセクションを配置する
2. 既存のコメント（// や /* */ ）はすべて保持する
3. セクション区切りのコメント（// ========= ）を追加・更新する
4. コードの動作は一切変更しない
5. 同一セクション内の順序は維持する

### 禁止事項
- 既存の処理コメントを削除しない
- 変数名や関数名を変更しない
- import文を勝手に追加・削除しない
- 不要な改行を追加しない

### セクション順序
1. 外部インポート
2. 内部インポート
3. ユーティリティ関数（存在する場合）
4. 初期設定
5. 状態管理
6. 算出プロパティ
7. ライフサイクル
8. メソッド

### 最終確認
整理後、「順序チェックリスト」の各項目が正しい順序で配置されているか確認してください。
